# Fortune-Teller

## Client Side Setup/Start

- npx create-vite
- cd client
- npm install
- npm run dev (to start the client)
- npm install axios

## Server Side Python Virtual Environment Setup

- python3 -m venv venv
- source venv/bin/activate
- pip3 install -r requirements.txt
- python3 main.py (to run the server)
